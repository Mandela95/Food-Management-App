live-server: https://food-management-app.netlify.app

Food Management System with two types of users (Admin-User) with different permissions for both roles.

The app enables role-based users to manage recipes, categories, and personal favorites efficiently. It provides secure authentication and a fully responsive layout optimized for mobile devices. Users can seamlessly view, create, edit, and organize recipes and categories, including image uploads with live previews, all within a clean and intuitive user interface.
